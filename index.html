CodeMate

终端里的 AI 编程搭档 — 支持多模型、自动备份、中英文双语

📝 Python 🔒 MIT License ⬇️ 3 Downloads GitCode 仓库

✨核心特性

🧠

多模型支持

支持 DeepSeek、OpenAI GPT-4o、Anthropic Claude、本地 Ollama 等多种大语言模型，灵活切换。

💾

自动备份

每次文件修改前自动备份原始文件，支持多轮次备份，防止代码丢失，安全无忧。

🌐

中英双语

完整的中英文界面支持，通过 /lang 命令一键切换，满足不同用户需求。

🛠️

强大工具集

内置文件读写/搜索、Shell 执行、网络搜索等工具，让 AI 直接操作你的项目文件。

📦

项目索引

自动建立项目结构索引，AI 快速理解代码库，提供更精准的上下文感知 assistance。

⚡

ReAct 架构

采用 Reason + Act 循环模式，AI 能够自主推理、调用工具、验证结果，完成复杂编程任务。

🚀快速开始

# 1. 安装依赖
pip install rich prompt_toolkit httpx pydantic pydantic-settings pyyaml
pip install -e .

# 2. 配置 codemate.toml（填入你的 API Key）

# 3. 启动
python -m codemate

⚙️项目结构

codemate/
├── __main__.py              # 程序入口
├── cli.py                   # 终端交互界面
├── i18n.py                  # 国际化 (中/英文)
├── config/
│   └── settings.py          # 配置管理 (TOML + 环境变量)
├── llm/
│   ├── providers.py         # 服务商工厂
│   ├── openai_compat.py     # OpenAI 兼容 API
│   └── anthropic_provider.py # Anthropic Claude API
├── agent/
│   ├── agent.py             # Agent 主循环
│   ├── context.py           # 对话上下文管理
│   └── tool_runner.py       # 工具调度执行
├── tools/
│   ├── file_tools.py        # 文件读写/搜索
│   ├── shell_tools.py       # Shell 执行
│   ├── web_tools.py         # 网络搜索
│   └── backup.py            # 文件备份管理
└── project/
    └── indexer.py           # 项目结构索引
