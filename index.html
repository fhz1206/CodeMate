核心功能

CodeMate 把 AI 能力带到你的终端，像搭档一样帮你写代码

🤖

多模型支持

兼容 DeepSeek、OpenAI、Anthropic Claude、Ollama 等任意 OpenAI 兼容 API，通过 TOML 配置文件自由切换。

📂

文件操作

读取、写入、编辑、搜索文件和目录，AI 直接操作你的代码库，像有经验的开发者一样高效。

💻

Shell 执行

在终端中直接运行命令、执行测试、管理 Git 提交，AI 会读取输出并据此调整下一步操作。

🔍

代码理解

项目索引、符号搜索、引用追踪、代码审查 — AI 深入理解你的项目结构。

🔒

自动备份

每次修改文件前自动备份原始内容，按对话轮次组织，超出配置数量自动清理，放心让 AI 改代码。

🌐

中英文双语

内置中英文双语支持，输入 `/lang zh` 或 `/lang en` 即可即时切换，LLM 也会用对应语言回复。

快速开始

三步启动你的 AI 编程搭档

1️⃣ 安装

pip install rich prompt_toolkit httpx
pip install -e .

2️⃣ 配置 codemate.toml

# 项目根目录下创建 codemate.toml
default_provider = "deepseek"
language = "zh"

[providers.deepseek]
type = "openai"
api_key = "sk-你的Key"
model = "deepseek-chat"
base_url = "https://api.deepseek.com"

3️⃣ 启动

python -m codemate

命令一览

输入 / 后按 Tab 自动补全

/help 显示帮助
/clear 清除对话历史
/status 查看项目状态和配置
/providers 查看 codemate.toml 配置
/lang zh 切换中文
/lang en 切换英文
/history 查看对话历史
/exit 退出

自动备份

AI 修改文件前的安全网

.codemate/
└── backups/
    ├── round_1/
    │   └── src/main.py  ← 修改前的原始文件
    ├── round_2/
    │   └── src/main.py
    └── round_3/

每次对话自动创建新轮次，超出 backup_rounds 配置的旧备份自动清理
