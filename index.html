核心功能

CodeMate 把 AI 能力带到你的终端

🤖

多模型支持

兼容 DeepSeek、OpenAI、Anthropic Claude、Ollama、阶跃星辰等任意 OpenAI 兼容 API，通过 TOML 配置文件自由切换。

📂

文件操作

读取、写入、编辑、搜索文件和目录，AI 直接操作你的代码库，修改前自动备份，支持 diff 对比。

💻

Shell 执行

在终端中直接运行命令、执行测试、管理 Git 提交，AI 会读取输出并据此调整下一步操作。

🧠

记忆系统

支持项目级和全局级持久化记忆，跨会话保留用户偏好和项目决策，重启不丢失。

🔒

自动备份 + Diff

每次修改文件前自动备份，支持 /diff 命令查看修改前后的红绿对比，安全放心。

🌐

中英文双语

内置中英文双语支持，输入 /lang zh 或 /lang en 即时切换，LLM 也会用对应语言回复。

📋

任务列表

AI 自动创建任务清单，逐步执行并标记完成，所有任务记录保存为 .md 文件。

🔄

子代理委派

主 Agent 可创建子代理执行特定任务（如代码审查），子代理有独立上下文，完成后主 Agent 审查整合。

快速开始

三步启动你的 AI 编程搭档

1️⃣ 安装

pip install rich prompt_toolkit httpx
pip install -e .

2️⃣ 配置 codemate.toml

# 项目根目录下创建 codemate.toml
default_provider = "deepseek"
language = "zh"

[providers.deepseek]
type = "openai"
api_key = "sk-你的Key"
model = "deepseek-chat"
base_url = "https://api.deepseek.com"

3️⃣ 启动

python -m codemate

4️⃣ 打包为 .exe

# 双击 build_exe.bat 即可
# 或手动执行:
pyinstaller --onedir --name codemate --console --add-data "codemate;codemate" --add-data "version.json;." codemate/__main__.py

命令一览

输入 / 后按 Tab 自动补全

/help显示帮助
/clear清除对话历史和缓存
/status查看项目状态和配置
/providers查看 codemate.toml 配置
/lang zh切换中文
/lang en切换英文
/history查看对话历史
/tool N查看工具调用完整输出
/diff N查看文件修改前后对比
/update检查并应用更新
/exit退出
